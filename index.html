Featured Project

IoT based Weather Forecast System using time series forecasting

A IoT-based weather station was developed that can remotely record the data and send it to the cloud database. We used ARIMA model to predict future values of the parameters and Extra trees classifier to classify the weather based on the predicted sensor values from the data from temperature sensor, humidity sensor, wind speed sensor, rain measurement sensor and pressure sensor. I worked on the Data science part of the project and optimized the hyper parameters and improved the model's accuracy recording weather data. Statistical methods like median were used to improve accuracy and also algorithm was such that the redundant data is not logged. Data was transferred to a Firebase real-time database remotely. An android app was developed that displayed the data as well the predicted weather.

  • Proteus
  • Machine learning
  • Android
  • Python
  • Firebase
